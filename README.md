# CodeMentor AI - The Developer's AI Mentor

A production-grade AI platform built with FastAPI and Llama-3.3.

## Features
- **Resilient AI Engine**: Modular architecture with retries and fallbacks.
- **Smart Analysis**: Deep syntactic and architectural insights.
- **AI Mentor**: Chat with an AI that understands your code context.
- **Premium UX**: Skeleton loaders, rotating tips, and real-time status.

## Tech Stack
- **Backend**: FastAPI, SQLAlchemy, Groq SDK.
- **Frontend**: TailwindCSS, Vanilla JS, Monaco Editor.
- **AI Model**: Llama-3.3-70b-versatile.
